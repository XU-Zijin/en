    Change website icon in hexo

    Tools hexo
    Created At : 2024-01-27 18:01
    Count:184 Views 👀 :
    1. 修改hexo的网站图标
      1. 在Next主题中更改
      2. 在3-hexo主题中更改
    2. 网站图标下载

    修改hexo的网站图标

    在Next主题中更改

    先把图标放进hexo/themes/主题/source/里面

    在/themes/主题/_config.yml中修改成自己的图标

    把这一项更改为你自己的图标名

    在3-hexo主题中更改

    先把图标放进hexo/themes/主题/source/img/里面

    在/themes/主题/_config.yml中修改成自己的图标

    把这项更改为你自己的网站(.jpg图片)图标。

    网站图标下载

    可以从阿里巴巴矢量图标库中免费下载网站图标，挑一个自己喜欢的即可(注:next主题要转换成.ico文件，3-hexo主题要.jpg文件)，或者你也可以自己绘制一个。
