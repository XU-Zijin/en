    Changing the Hexo Site Icon

    How to Change in the Next Theme

    First, place your icon into the hexo/themes/theme/source/ directory.

    Then modify the icon setting in your theme’s _config.yml like so:

    favicon: /favicon.ico

    Change this line to the name of your own icon.

    How to Change in the 3-Hexo Theme

    First, place your icon into the hexo/themes/theme/source/img/ directory.

    Then modify the icon setting in your theme’s _config.yml like so:

    favicon: /img/favicon.jpg

    Change this line to your personal website icon (.jpg image).

    Website Icon Download

    You can download free website icons from Alibaba’s Vector Icon Library. Just pick one you like (note: the Next theme requires an .ico file, while the 3-hexo theme uses a .jpg file), or you can also create your own.
